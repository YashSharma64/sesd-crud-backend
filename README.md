# SESD CRUD Backend (TypeScript)

A clean, layered CRUD backend built using **Node.js, Express, and TypeScript** following
**OOP principles** and **controllers → services → repositories** architecture.

## Tech Stack
- Node.js
- Express.js
- TypeScript
- MongoDB (Mongoose)

## Architecture
- Controllers
- Services
- Repositories
- Models

## Status
Project setup initialized. CRUD implementation in progress.
